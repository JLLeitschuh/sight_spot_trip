# sight_spot_trip

### Prerequisites

1. [maven][maven]
2. [bower][bower]
3. [neo4j][neo4j]

### How to run

1. download `neo4j-ce`, start it and set password to `secret`
2. git clone https://github.com/OrangeChocolate/sight_spot_trip.git
3. cd sight_spot_trip
4. bower install
5. mvn spring-boot:run
6. navigate http://localhost:1234/
